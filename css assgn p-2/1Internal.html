Internal CSS

An internal CSS is used to define a style for a single HTML page.Internal CSS is one of the most widely used CSS forms for changing, styling, and modifying the unique styles of a single web page. You can use the internal CSS by integrating the style element in the head section of a HTML web page. Internal CSS can be applied to the whole web page but not on multiple web pages and you can style several web pages by using the same code on every page.
